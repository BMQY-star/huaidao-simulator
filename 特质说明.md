# 特质说明

本文件说明学生特质库的结构与分类。特质仅用于角色表现与导师指导结算时的间接影响，不直接修改初始数值。

## 1. 分类
- **主特质（人格型）**：每名学生仅 1 个。
- **副特质（诙谐型）**：每名学生 2-3 个。
- **极性**：正向 / 负向 / 中性，用于 UI 颜色区分与权重抽取。

## 4. 规则要点
- 采用 60/20/20（正/负/中）权重抽取。
- 冲突特质不会同时出现。
- 主特质显示在学生卡片顶部，副特质显示在论文进度上方。
